Documentation du langage SIO

Syntaxe :

Begin : Début de bloc, code ou boucle.

End :  fin de bloc, code ou boucle.

LET <VARIABLE> <Entier> : Attribue un entier a la variable.

ADD <terme> <terme> <VARIABLE> : Additionne les deux termes (Entier ou variable) dans une variable.

DIV <terme> <terme> <VARIABLE> : Divise les deux termes (Entier ou variable) dans une variable.

MOD <terme> <terme> <VARIABLE> : Additionne les deux termes (Entier ou variable) dans une variable.

INC <VARIABLE> : Incrémente la variable (+1).

IF  <terme> <condition> <terme> : Compare les deux termes (Entier ou variable) et s'exécute si positif.

WHILE <terme> <condition> <terme> : S'exécute tant que la comparaison des deux termes (Entier ou variable) est positive.

WRITE <VARIABLE> : Renvoi la valeur contenue dans la variable.
